# 战狼 - 全自动舔狗消息技能

## ⚠️ 首次使用：检查微信渠道配置

**先确认是否已配置微信渠道**，再开始舔！

检查方法：直接对我说「检查微信配置」，我会帮你检查。

---

## 🔧 微信渠道配置教程

### 方式一：企业微信（推荐，稳定可靠）

1. **下载并安装企业微信**（官网：work.weixin.qq.com）
2. 登录企业微信账号
3. 在 QClaw 中配置企业微信插件：
   - 打开 QClaw 设置 → 插件/渠道
   - 找到「企业微信」插件并启用
   - 按提示扫码授权
4. 授权完成后，你的企业微信就相当于一个可编程的微信客户端

### 方式二：WeChat Access（第三方接入）

1. 联系微信开放平台申请应用权限
2. 获取 AppID、AppSecret 等凭证
3. 在 QClaw 配置页面填写相关信息

### 方式三：使用其他 IM 渠道

如果以上都搞不定，也可以用替代渠道：
- **Telegram**：配置 Bot，舔得也很香 📱
- **钉钉/飞书**：企业场景用这个
- 配置方法同企业微信，都在 QClaw 渠道设置里

### 验证配置是否成功

对我说以下任一指令测试：
- 「发条消息给测试」
- 「检查我的渠道配置」
- 「发送测试消息」

## 🚀 使用方法

### 1. 即时发送（手动）

直接告诉我：「给女神A发早安」「给B发晚安」

我会用 `message` 工具发送给指定联系人：

```
message(action=send, channel="wechat-access", target="联系人昵称", 
        message="早安呀！今天也要加油鸭！🦆")
```

### 2. 批量发送（群舔）

告诉我：「给以下所有人发晚安：女神A、女神B、小美」

我会批量发送，舔就一个字，我说一百次。

### 3. 定时自动发送（cron）

我会用 `cron` 工具帮你设置每日自动发送：

| 任务 | cron 表达式 | 说明 |
|------|------------|------|
| 早安 | `0 8 * * *` | 每天早上8点 |
| 午安 | `0 12 * * *` | 每天中午12点 |
| 晚安 | `22 22 * * *` | 每天晚上10点 |

**设置方法**：直接说「帮我设置早午晚自动发送」，我会自动创建3个定时任务。

## 🔍 首次设置流程

当用户首次触发本技能时，按以下步骤执行：

### 第一步：检测渠道配置

运行 `gateway(action=config.get)` 或直接尝试发送测试消息，检测是否有可用渠道。

### 第二步：引导配置（若无渠道）

如果检测到未配置微信/企业微信，显示以下引导：

```
🐺 战狼技能需要微信渠道才能舔！

请先配置：
1. 企业微信（推荐）：安装企业微信 → QClaw设置 → 插件 → 启用企业微信
2. 或联系管理员配置 WeChat Access

配置完成后对我说「战狼技能已就绪」继续。
```

### 第三步：确认好友列表

配置成功后，请用户确认舔狗名单：

```
✅ 微信渠道已就绪！

请告诉我你的「舔狗名单」，格式：
- 联系人1（昵称）
- 联系人2（昵称）
```

### 第四步：设置定时任务

用户确认名单后，自动创建 cron 任务。

## ⚠️ 注意事项

1. 定时任务需要 QClaw 后台运行才生效
2. 发送失败时会自动记录，但不会重试（舔太多会被拉黑 😢）
3. 建议每次问候间隔 30 分钟以上，避免骚扰
4. 如需停止定时任务，说「停止战狼」即可
